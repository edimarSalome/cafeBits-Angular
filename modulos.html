Módulos

app.js

                            
// Declaração do módulo
var app = angular.module('cafeBits', []);

 

modulos.html

                            
<!--- referência ao JS da aplicação --->
<script src="js/app.js"></script>

...

<!--- Inicia o app(módulo) do Angular JS --->
<body ng-app='cafeBits'>
...
</body>

Observação

É possível informar dependência entre os módulos, de forma que um conjunto de módulos se unem para formar uma aplicação e quando necessário, pode ser substituído ou atualizado com baixo impacto.
